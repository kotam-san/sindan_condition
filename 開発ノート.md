## 技術スタック

- **フロントエンド**：単一HTMLファイル（React 18 CDN + Babel Standalone）
- **バックエンド**：Cloudflare Worker（`worker/src/index.js`）
- **AI提案生成**：Claude API（claude-haiku-4-5-20251001）
- **フォールバック**：軸別静的テキスト（`SUGGESTIONS` 定数）
- **スタイル**：インラインスタイル（ビルド不要）
- **チャート**：SVG自前実装（外部ライブラリなし）
- **PDF保存**：`window.print()` + `@media print` CSS

## 仕様変更の記録

### PDF直接ダウンロード → Gated Content + AI提案に再設計

当初：PDFのみで提案を静的表示（メール不要）

現在：メアド入力ゲートを設け、ゲート後にClaudeが生成した個別提案を表示。PDFもゲート後に提供。

## 次にやること

1. **Claude Codeを再起動**してGitHub MCPを有効化
2. **GitHubリポジトリを作成**してコードをpush
3. **Cloudflare PagesをGitHubと連携**（pushで自動デプロイ）
4. **Cloudflare Workerをデプロイ**
   ```sh
   cd worker
   wrangler secret put ANTHROPIC_API_KEY
   wrangler deploy
   ```
5. **WORKER_URL を index.html に設定**（デプロイ後に表示されるURL）
6. **CTAリンクを差し替え**（`href="#"` → 無料相談ページのURL）
7. **note記事の執筆・公開** → `NOTE_ARTICLES` のURLを設定
